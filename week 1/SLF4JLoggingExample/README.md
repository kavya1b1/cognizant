# SLF4J Logging Example

## Description

This project demonstrates how to use the SLF4J logging framework with Logback to log warning and error messages in a Java application.

---

## Technologies Used

- Java
- Maven
- SLF4J
- Logback

---

## Project Structure

```
src/
└── main/
    └── java/
        └── LoggingExample.java
```

---

## Dependencies

- slf4j-api 1.7.30
- logback-classic 1.2.3

---

## Output

```
ERROR LoggingExample - This is an error message
WARN  LoggingExample - This is a warning message
```

---

## Concepts Covered

- Logging using SLF4J
- LoggerFactory
- Error Logging
- Warning Logging
- Logback Integration
