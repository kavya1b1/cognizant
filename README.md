# Week 1 – Java FSE Exercises

This folder contains the solutions for **Week 1** of the **Cognizant Digital Nurture 4.0 – Java Full Stack Engineer (FSE)** training program.

The exercises focus on Java Design Patterns, Searching Algorithms, Recursion, Unit Testing, Mocking Frameworks, and Logging.

## Exercises

### 1. Singleton Pattern Example

**Objective**

Implement the **Singleton Design Pattern** to ensure that only one instance of a class exists throughout the application.

**Concepts Covered**

- Singleton Design Pattern
- Private Constructor
- Static Instance
- Lazy Initialization
- Object Reusability

---

### 2. Factory Method Pattern Example

**Objective**

Implement the **Factory Method Design Pattern** to create different document types such as Word, PDF, and Excel documents.

**Concepts Covered**

- Factory Method Pattern
- Abstract Classes
- Inheritance
- Polymorphism
- Object Creation

---

### 3. E-Commerce Search Example

**Objective**

Implement searching functionality for an e-commerce platform using Linear Search and Binary Search.

**Concepts Covered**

- Big O Notation
- Linear Search
- Binary Search
- Time Complexity Analysis
- Best, Average, and Worst Cases

---

### 4. Financial Forecasting Example

**Objective**

Develop a recursive solution to predict future financial values based on historical growth rates.

**Concepts Covered**

- Recursion
- Recursive Algorithms
- Time Complexity
- Optimization Techniques

---

### 5. JUnit Setup Example

**Objective**

Configure JUnit and implement basic unit testing features.

**Exercises Included**

- JUnit Setup
- Assertions
- Arrange-Act-Assert (AAA) Pattern
- Setup and Teardown Methods

**Concepts Covered**

- Unit Testing
- Assertions
- Test Lifecycle
- Test Organization

---

### 6. Mockito Example

**Objective**

Learn mocking and interaction verification using Mockito.

**Exercises Included**

- Mocking and Stubbing
- Verifying Method Interactions

**Concepts Covered**

- Mock Objects
- Stubbing
- Verification
- Dependency Isolation

---

### 7. SLF4J Logging Example

**Objective**

Demonstrate logging in Java applications using **SLF4J** with **Logback**.

**Concepts Covered**

- Logger
- LoggerFactory
- Warning Logs
- Error Logs
- Logging Best Practices
